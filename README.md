# Dashboard de Gestión de Citas Médicas

## Descripción

Este proyecto fue desarrollado para el curso de **Inteligencia de Negocios**. El dashboard permite analizar información de citas médicas mediante gráficos interactivos utilizando Python y Streamlit.

## Objetivos

- Analizar el comportamiento de las citas médicas.
- Identificar las especialidades con mayor demanda.
- Visualizar el estado de las citas (atendidas, canceladas y no asistidas).
- Mostrar la cantidad de citas por mes.

## Tecnologías utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- NumPy
- Scikit-Learn

## Archivos del proyecto

- `app.py` → Aplicación principal del dashboard.
- `citas_medicas.csv` → Base de datos simulada con 5000 registros.
- `requirements.txt` → Dependencias necesarias para ejecutar la aplicación.
- `README.md` → Documentación del proyecto.

## Ejecución local

1. Instalar las dependencias:

```bash
pip install -r requirements.txt
```

2. Ejecutar la aplicación:

```bash
streamlit run app.py
```
